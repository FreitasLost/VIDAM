Regras do Servidor

Regras Fundamentais

• Anti-RDM: Proibido matar sem motivo RP

• Anti-VDM: Proibido usar veículos como arma

• Anti-CK: Character Kill só com autorização

• Anti-MG: Proibido usar poderes irreais

• Anti-PG: Mantenha ações realistas

• Anti-Combat Logging: Não deslogue em ação

Regras de Interpretação

• Mantenha o personagem sempre no personagem

• Use /me para ações

• Use /do para narração

• Interprete ferimentos

• Respeite tempo de recuperação

• Mantenha coerência nas ações

Regras de Interação

• Sem MetaGaming (usar info fora do RP)

• Sem PowerGaming (ações irreais)

• Respeite SafeZones

• Sem Breaking Character

• Respeite limitações físicas

• Mantenha diálogos realistas

Regras de Combate

• Avise assaltos

• Respeite tempo de rendição

• Sem camping em spawns

• Siga tempos de respawn

• Use armas adequadamente

• Mantenha distância realista

Regras de Facções

• Respeite territórios

• Mantenha hierarquia

• Use uniformes apropriados

• Siga códigos internos

• Respeite alianças/rivalidades

• Mantenha ações organizadas

Regras de Comunicação

• Use chat local para proximidade

• Celular para longas distâncias

• Rádios apenas com item

• Sem gritos desnecessários

• Mantenha língua adequada

• Respeite canais oficiais
